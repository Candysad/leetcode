### 翻转链表

- $O(1)$ 空间复杂度翻转链表
- 将链表倒车

```python
def reverse_chain(head: Optional[ListNode]) -> Optional[ListNode]:
    if head is None:
        return None

    sentinel = ListNode(0, next=head)
    now = head
    next = now.next
    while next is not None:
        t = next.next
        next.next = now
        sentinel.next = next

        now = next
        next = t

    head.next = None
    return sentinel.next
```
