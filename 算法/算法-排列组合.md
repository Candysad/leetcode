$A_n^m = \frac{n!}{m!}$ 





### 组合

$n$ 个物品， 选 $m$ 个，不区分顺序

$C_n^m = \frac{n!}{(n-m)!m!} $ 

```python
def Combination(n:int, m:int):
    result = 1
    for i in range(m, m-n, -1):
        result *= i
    for i in range(1, n+1):
        result //= i
    return result
```
