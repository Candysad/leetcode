# 背包问题

选可选项放入背包，形成组合，每个选项选一次或多次



### 单次选，数可重复

- 从小到大房放数
- 但是放的时候倒着看，避免自己先出现在更小的位置使自己被重复算

nums 中每个数取一次，问组成和为 k 的情况有多少种

```python
def backpack(nums, k: int) -> int: # K为加和的目标
    nums.sort()
    dp = [0]*(k+1)
    for num in nums:
        if num > k:
            break
        
        for i in range(k, num-1, -1):
            dp[i] += dp[i-num]
        dp[num] += 1
    print(dp)
    return dp[k]
```
